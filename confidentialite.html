Règles de confidentialité L’application « Sur les Ailes de la Foi » est éditée par Alex Léonel DJOKO. Le terme « Application » désigne l’application mobile Android « Sur les Ailes de la Foi ». Nous accordons une grande importance à la protection de votre vie privée. Cette politique explique quelles données peuvent être collectées, comment elles sont utilisées et avec qui elles peuvent être partagées. 1. Données collectées Contrairement à une utilisation purement locale, certaines données peuvent être transmises lorsque vous utilisez certaines fonctionnalités de l’Application. a) Données fournies par l’utilisateur Lorsque vous envoyez un message via la fonctionnalité de retour (feedback), nous collectons : * le contenu du message * la catégorie associée Ces données sont fournies volontairement par l’utilisateur. b) Données techniques et appareil Lors de l’envoi d’un feedback, les informations suivantes peuvent être collectées automatiquement : * modèle de l’appareil * version du système Android * version de l’application * date et heure (timestamp) Ces données permettent d’améliorer l’application et de mieux comprendre les retours utilisateurs. 2. Utilisation des données Les données collectées sont utilisées uniquement pour : * traiter les retours utilisateurs * améliorer les fonctionnalités et la stabilité de l’application 3. Partage des données Les données peuvent être transmises à des services tiers utilisés par l’application, notamment : * Google Firebase (hébergement et base de données) Ces services traitent les données conformément à leurs propres politiques de confidentialité. Aucune donnée n’est vendue à des tiers. 4. Publicité et identifiants L’application peut utiliser des services Google pouvant collecter des identifiants publicitaires (Advertising ID), notamment pour : * afficher des publicités * mesurer leur performance Ces données peuvent être utilisées conformément aux règles de Google. 5. Consentement (RGPD) Selon votre localisation, un consentement peut être demandé avant l’utilisation de certaines fonctionnalités (notamment publicitaires) via la plateforme de gestion du consentement de Google. 6. Sécurité des données Les données sont transmises de manière sécurisée via des protocoles chiffrés (HTTPS). 7. Conservation des données Les données envoyées (feedback) sont conservées uniquement pendant la durée nécessaire à leur traitement. 8. Vos droits Conformément aux réglementations en vigueur, vous pouvez demander la suppression de vos données en contactant : app.surlesailesdelafoi@gmail.com 9. Modifications Cette politique peut être mise à jour à tout moment. Toute modification sera publiée dans cette section. 10. Contact Pour toute question concernant cette politique : app.surlesailesdelafoi@gmail.com
